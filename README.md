# parkOrl
ParkOrl est un site web développé pour fournir des informations sur les disponibilités des places de parking à Orléans. Ce projet a été réalisé lors de mon immersion professionnelle de 35h, sous la supervision d’un tech lead, dans le cadre de ma reconversion vers le développement web. L’objectif principal était l’apprentissage de JavaScript, le CSS ayant été ajouté ensuite pour améliorer l’apparence du site.

Objectifs et apprentissages :

Travailler avec Git : push, commit, versioning.

Appliquer et expérimenter JavaScript pour rendre le site interactif.

Publier un projet concret sur GitHub Pages.
